# BookMind 0.1.0

BookMind 的首个公开桌面版本。

## 主要内容

- 本地书库导入与阅读，支持 TXT、Markdown、EPUB、PDF、MOBI 等格式。
- 阅读进度、书签、高亮、批注与知识整理均默认保存在本机。
- 本地全文检索、人物档案与关系图谱、任务中心和数据备份。
- 可选配置兼容 OpenAI API 的 AI 服务；只有主动配置后才会向对应服务发送内容。
- 提供简体中文、English、日本語、Español、Français 和 한국어界面。

## 安装

- Windows：下载 `.msi` 或 `.exe` 安装包。
- macOS：下载并打开 `.dmg`，将 BookMind 拖入“应用程序”。
- Linux：下载适合发行版的 `.AppImage`、`.deb` 或 `.rpm` 包。

每个平台的 `SHA256SUMS-*.txt` 文件包含对应安装包的 SHA-256 校验值。请在安装前核对下载文件的校验值。

## 使用说明

本项目不包含书籍、索引或用户数据。请只导入拥有合法使用权的内容。云端 AI 与翻译服务为可选功能；使用前请确认服务提供方的隐私政策。
